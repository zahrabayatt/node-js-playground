import data in database using this command:

mongoimport --db mongo-exercises --collection courses --file exercise-data.jso --jsonArray

if you run this command and get error like:
'mongoimport' is not recognized as an internal or external command,operable program or batch file.

you need to download the MongoDB Command Line Database Tools Download: https://www.mongodb.com/try/download/database-tools

and Then follow this tutorial on how to install it: https://www.mongodb.com/docs/database-tools/installation/installation-windows/
